## Angular packages installation

Use the command on {workspace}/GameShop/GameShop.UI/ClientApp

```bash
npm install 
```

## ASP.NET Core packages installation

Use the command on {workspace}/GameShop/GameShop.UI

```bash
dotnet run
```

## Running appliaction

Use the command on {workspace}/GameShop/GameShop.UI

```bash
dotnet run
```
